HTML Cheat Sheet

Basic Structure

<!DOCTYPE html> <html> <head> <title>Page Title</title> </head> <body> Content </body> </html>

Defines the basic structure of an HTML document

Headings and Paragraphs

<h1> to <h6>

Defines headings of different levels

<p>

Defines a paragraph

Links and Lists

<a href="url">Link Text</a>

Creates a hyperlink

<ul> <li>Item 1</li> <li>Item 2</li> </ul>

Defines an unordered list

Images and Forms

<img src="image.jpg" alt="Description">

Embeds an image

<form> <input type="text"> <button>Submit</button> </form>

Creates a form with input and button

Structural Elements

<div>Content</div>

Defines a division or a section in an HTML document

<span>Text</span>

Inline container used to apply styles to a specific part of the text

Semantic Elements

<header>Header Content</header>

Defines the header of a document or a section

<footer>Footer Content</footer>

Defines the footer of a document or a section

<nav> <a href="#">Home</a> <a href="#">About</a> </nav>

Defines a set of navigation links

Tables and Entities

<table> <tr> <th>Header</th> <th>Header</th> </tr> <tr> <td>Row 1</td> <td>Row 1</td> </tr> </table>

Creates a table with rows and columns

&lt; for < &gt; for > &amp; for & &copy; for ©

Used to display special characters in HTML
